### **Part 1: Key Definitions**

#### **Introduction to Data Science (Chapter 1)**

* 
**Data Science:** The science of understanding data using processes, tools, and techniques (like identifying, collecting, and exploring data) to aid in decision-making.


* 
**Big Data:** Large volumes of data defined by the 4 Vs: **Volume** (amount), **Velocity** (speed of generation), **Variety** (different formats), and **Veracity** (uncertainty/quality) .


* 
**Machine Learning (ML):** A subset of AI where the model learns on its own without human intervention.


* 
**Deep Learning:** A subset of ML that works effectively on larger datasets and aims at pattern recognition by imitating the human brain.


* 
**Descriptive Analytics:** Analysis that answers "What has happened?".


* 
**Predictive Analytics:** Analysis that answers "What will happen?".



#### **Exploratory Data Analysis (Chapter 3)**

* 
**Continuous Variable:** Quantitative variables that can take infinite values and can be measured (e.g., height, weight).


* 
**Categorical Variable:** Variables that can be categorized into distinct, finite groups (e.g., gender, pass/fail).


* 
**Boxplot:** A plot that divides data into quartiles and is used to identify outliers and skewness.


* 
**Standardization (Z-score):** A scaling technique calculated as .


* 
**Normalization:** A scaling technique calculated as .



#### **Data Visualization (Chapter 4)**

* 
**Data Visualization:** The presentation of quantitative data using visually appealing images to help derive insights, identify patterns, and analyze trends.


* 
**Infographics:** A visual representation best for telling a premeditated story to guide the audience to specific conclusions.


* 
**Heatmap:** A visualization where values are represented by color intensity; lighter shades often indicate higher correlation in correlation matrices.



#### **Dimensionality Reduction (Chapter 5)**

* 
**Multicollinearity:** A state where there is a high correlation between independent variables, which can affect model accuracy.


* 
**Eigen Values:** A measure indicating the variance of variables; used to determine the number of factors in Factor Analysis (values > 1 are typically selected).


* 
**Scree Plot:** A plot used to determine the optimal number of factors or components based on eigen values.



#### **Supervised vs. Unsupervised Learning (Chapter 6)**

* 
**Supervised Learning:** Algorithms that learn under the presence of a "supervisor" using labeled data (input and output are known).


* 
**Unsupervised Learning:** Algorithms used for exploration on unlabeled data to mine rules, detect structures, or summarize data without defined outcomes.


* 
**Regression:** A supervised technique to determine the statistical relationship between variables where the output is numeric/continuous.


* 
**Classification:** A supervised technique used when the output variable is categorical (e.g., Yes/No).



#### **Unsupervised Techniques (Chapter 7)**

* 
**Clustering:** An unsupervised technique that groups similar data items together based on a similarity metric.

#### **Regression (Chapter 9)**

* 
**Linear Regression:** A model that uses a straight line () to explain the relationship between a dependent variable and independent variable(s).

### **Part 3: Viva Related Questions & Answers**

**Q1. Differentiate between Regression and Classification.**

* 
**Answer:** Regression is used when the output variable is numeric/continuous (e.g., predicting sales volume), whereas Classification is used when the output variable is categorical (e.g., predicting Yes/No for a loan default).



**Q2. Explain the difference between Supervised and Unsupervised Learning.**

* **Answer:** Supervised learning uses labeled data (input and output are known) to train a model (e.g., Regression). Unsupervised learning uses unlabeled data to find hidden patterns or structures without a predefined outcome (e.g., Clustering).



**Q3. What is the difference between Infographics and Data Visualization?**

* **Answer:** Infographics are subjective and tell a premeditated story to guide the audience to a specific conclusion. Data visualization is objective, automatically generated, and allows the audience to draw their own conclusions.



**Q4. How do you handle Multicollinearity?**

* **Answer:** Multicollinearity occurs when independent variables are highly correlated. It affects model accuracy. It is handled by applying dimensionality reduction techniques (like Factor Analysis) to reduce the number of variables.



**Q5. What is the purpose of a Confusion Matrix?**

* **Answer:** It is used to evaluate the performance of a classification model. It calculates metrics like Accuracy, Sensitivity (True Positive Rate), Specificity (True Negative Rate), and Precision based on True Positives, False Positives, True Negatives, and False Negatives.



**Q6. Explain the concept of Random Forest.**

* **Answer:** Random Forest is an ensemble technique that uses multiple decision trees. For classification problems, it uses voting (the class with max votes wins), and for regression, it calculates the average of the trees' outputs.



**Q7. What is the Apriori Algorithm used for?**

* **Answer:** It is an algorithm used for Association Rule Mining. It identifies frequent itemsets in a dataset (e.g., market basket analysis) by using metrics like Support and Confidence to generate rules like "If Item A is bought, Item B is likely to be bought".



**Q8. What is the difference between Agglomerative and Divisive Clustering?**

* **Answer:** Agglomerative is a "bottom-up" approach where each record starts as its own cluster and they merge until one cluster remains. Divisive is a "top-down" approach where all records start in one cluster and are split into smaller clusters.



**Q9. What is a Scree Plot used for in Factor Analysis?**

* **Answer:** A Scree Plot visualizes the Eigen values against the number of factors. It helps determine the optimal number of factors to retain by looking for the "elbow" or the point where the curve flattens (usually factors with Eigen value > 1).



**Q10. Why is Naive Bayes called "Naive"?**

* 
**Answer:** It is called "Naive" because it assumes that all predictor (independent) variables are independent of each other, which is a strong and often unrealistic assumption in real-world data.



**Q11. What is the difference between Euclidean and Manhattan distance?**

* 
**Answer:** Euclidean distance is the straight-line distance between two points (), while Manhattan distance (City block) is the sum of the absolute differences of their coordinates ().



**Q12. What does AUC (Area Under the Curve) indicate in a ROC plot?**

* **Answer:** AUC represents the degree of separability. An AUC of 100% means the model perfectly distinguishes between positive and negative classes. An AUC of 50% means the model has no discrimination capacity (random guess).
